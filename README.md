# SEISMIC-IGNITED-LANDSLIDE-SUSCEPTIBILITY-MAPPING
A hybrid Deep Learning framework predicts landslide susceptibility using multi-temporal satellite data and seismic features.
Key Highlights:

✅ 4 Domains Covered: ML/DL, Remote Sensing & GIS, Signal Processing, Geology

✅ Innovative Approach: Fusion of static (geology/topography) and dynamic (seismic) data

✅ Real-World Impact: Improves earthquake-triggered landslide risk assessment

✅ Feasible: Uses publicly available datasets and open-source tools
