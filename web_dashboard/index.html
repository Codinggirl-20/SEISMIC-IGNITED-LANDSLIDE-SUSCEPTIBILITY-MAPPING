About the Model

This hybrid model operates by extracting a spatial neighborhood patch around each grid cell. A 2D CNN processes the topography & satellite bands, while an MLP ingests transient seismic signals computed via site-specific GMPEs. The fused vector predicts landslide triggering probability.
